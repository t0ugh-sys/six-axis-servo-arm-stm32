# 供电建议（MG996R + MG90S）

## 为什么一定要单独给舵机供电？

舵机启动/堵转时电流会瞬间飙升，如果和 MCU 共用弱电源，典型表现是：
- STM32 复位/死机
- PCA9685 输出抖动
- 舵机乱跳

## 推荐电源规格

以“能稳定做机械臂”为目标：
- 电压：**6.0V**（多数 MG996R / MG90S 在 6V 更有力，但更热）
- 电流：建议 **>= 10A**（更稳妥可选 15A/20A 级别）

## 必做的抗干扰措施

- 舵机电源与 STM32 **共地**
- PCA9685 `V+` 近端加大电容（1000uF~2200uF）
- 舵机供电线尽量短且更粗（降低压降）
- 机械臂动作要做“平滑插补”（固件层面），避免猛冲导致电流尖峰
