# 标定流程（建议每个关节都做）

目标：得到每个舵机通道的
- 中心点偏置（90 度是否真正居中）
- 正反向（角度增大时是顺时针还是逆时针）
- 安全脉宽范围（避免顶死、打齿、烧舵机）

## 建议步骤

1. 先“空载/不装连杆”单独插一个舵机在 PCA9685 通道上。
2. 固件先用安全范围（例如 500~2500us）+ 50Hz 频率。
3. 让舵机输出 1500us（理论中心），把舵盘装到你需要的“机械零位”。
4. 小步进测试：
   - 例如每次改 20us 或 1~2 度，观察是否卡住/顶死/发热。
   - 记录“刚好不顶死”的最小/最大 us，作为该关节限位。
5. 重复到 6 个关节，填入 `firmware/Inc/arm_config.h`。

## 经验值（仅作起点，最终以你的实际机械为准）

- MG996R：常见可用范围约 500~2500us（有些舵机更窄）
- MG90S：常见可用范围约 600~2400us（有些舵机更窄）
